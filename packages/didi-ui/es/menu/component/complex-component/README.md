# 复杂组件

- DragGroupView ： 拖拽组合组件
- SelectGroupView ： 单选组合组件

# API

#### DragGroupView ： 拖拽组合组件

| 标题 | 类型  | 默认值 | 描述     |
| ---- | ----- | ------ | -------- |
| data | nodes | " "    | 数据节点 |

#### SelectGroupView ： 单选组合组件

| 标题 | 类型  | 默认值 | 描述     |
| ---- | ----- | ------ | -------- |
| data | nodes | " "    | 数据节点 |
