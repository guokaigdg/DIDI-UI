# 菜单栏

- MainMenu 主菜单
- SubMenu 子菜单
- SubPopover 快捷键按钮(弹出子菜单)

## API

- MainMenu 主菜单
  | 标题 | 类型 | 默认值 | 描述 |
  | ---- | ----- | ------ | -------- |
  | data | nodes | | 节点数据 |

- SubMenu 子菜单
  | 标题 | 类型 | 默认值 | 描述 |
  | ---- | ----- | ------ | -------- |
  | data | nodes | | 节点数据 |

- SubPopover 弹窗按钮(弹出子菜单)
  | 标题 | 类型 | 默认值 | 描述 |
  | ---- | ----- | ------ | -------- |
  | children \* | node | | 快捷按钮 |
  | content | nodes | | 弹出的子菜单 |
