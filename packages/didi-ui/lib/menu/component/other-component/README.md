# 其他菜单组件

- FilterInputView 过滤输入框

- GroupTitleView 分组标题组件

- HintText 提示文本

- HintButton 提示按钮

## API

#### FilterInputView 过滤输入框

| 标题        | 类型   | 默认值 | 描述               |
| ----------- | ------ | ------ | ------------------ |
| placeholder | String | " "    | 搜索框默认显示文字 |
| value       |        |        |                    |

#### GroupTitleView 分组标题组件

| 标题 | 类型   | 默认值 | 描述         |
| ---- | ------ | ------ | ------------ |
| text | String | " "    | 分组标题内容 |

#### HintText 提示文本

| 标题 | 类型   | 默认值 | 描述         |
| ---- | ------ | ------ | ------------ |
| text | String | " "    | 提示文本内容 |

#### HintButton 提示按钮

| 标题 | 类型   | 默认值 | 描述         |
| ---- | ------ | ------ | ------------ |
| text | String | " "    | 提示按钮内容 |
