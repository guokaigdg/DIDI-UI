### 客户端 React 项目组件库

### 📦 安装

```
npm i solidoc-base-component-lib
```

```
cnpm i solidoc-base-component-lib
```

### 🔨 示例

- 引入样式

```js
import { Button } from "solidoc-base-component-lib";
ReactDOM.render(<DatePicker />, mountNode);
```

#### 🌰 例子

```js
import { SidebarSwitch } from "solidoc-base-component-lib";

<SidebarSwitch
  icon="🍉",
  title={"工作区"}
/>
```
