## React 项目组件库

#### 📦 安装

```
npm i DIDI-UI
```

```
cnpm i DIDI-UI
```

#### 🔨 示例

- 引入样式

```js
import { SidebarSwitch } from "DIDI-UI";
```

#### 🌰 例子

```js
<SidebarSwitch
  icon="🍉",
  title={"工作区"}
/>
```
