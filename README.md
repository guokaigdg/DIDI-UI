## React 项目组件库

### 📦 安装

```
npm i DIDI-UI
```

```
cnpm i DIDI-UI
```

### 🔨 示例

- 引入样式

```js
import { Button } from "DIDI-UI";
```

### 🌰 例子

```js
<Button
  onClick={() => {
    this.handleClick();
  }} //onClick
  disabled={false} //disabled
>
  登陆
</Button>
```
